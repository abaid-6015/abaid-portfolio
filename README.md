# Abaid-ul-Rehman Portfolio

A professional, modern portfolio website built with React + Vite featuring:
- 🎨 Custom animated cursor
- ✨ Particle/network canvas animation in hero
- 🌐 3D floating tech icon orbs
- 📊 Animated skill bars
- 💼 Interactive project showcase (7 projects)
- 📅 Timeline-based experience section
- 📧 EmailJS contact form (sends to abaidbse@gmail.com)
- 🚀 Fully responsive for all devices
- ⚡ Optimized for Vercel deployment

## Tech Stack
- React 18 + Vite
- Framer Motion animations
- Three.js (3D animations)
- EmailJS (contact form)
- React Type Animation
- CSS Variables design system
